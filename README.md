# `React` with `Webpack4`

## 사용법
### 환경 구성
`yarn` 혹은 `npm install`  

### 개발서버 실행
`yarn start` 혹은 `npm start`  

### 빌드
`yarn build` 혹은 `npm build`

### 배포
#### 준비
빌드 이후 firebase 설치가 되어있다는 가정하에  
`yarn fbinit && yarn fbdeploy`
